2.Ver todos los productos y realizar búsquedas por filtrado de nombre y ordenamiento por precio.
	EndPoint >/api/productos
	QueryParams
	{
		name : EJ. Mochila
		sort : EJ. true (ASC = true | DESC = false)
	}

4. Agregar un producto al carrito de compra.
	EndPoint >/api/carrito
	Body
	{
		"clientId": 123,
		"productId": 123,
		"amount": 123
	}
5. Debe permitir modificar el carrito de compra.
	EndPoint >/api/carrito
	Body
	{
		"clientId": 123,
		"productId": 123,
		"amount": 123
	}
6. Debe permitir eliminar un producto del carrito.
	EndPoint [DELETE] >/api/carrito/:clientId/:productId
7. Debe permitir efectuar una compra.
	EndPoint [POST] >/api/orden/:clientId
8. Agregar un reporte de balance por día, debe devolver la recaudación del día con las
ventas realizadas y sus respectivos productos.
	EndPoint [GET] >/api/orden
	QueryParams
	{
		from: 2022-09-19 (Fecha desde)
		to: 2022-09-20 (Fecha hasta)
	}
